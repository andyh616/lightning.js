# lightning.js

> Node.js client for the Lightning data visualization server.


## Install

```sh
$ npm install --save lightning.js
```


## Usage

```js
var Lightning = require('lightning.js');

var lightning = new Lightning();
lightning.line([1,1,2,3,5,8,13,21]);

```
